## Contributing Guide
- Clone the repo and install dependencies.

### Prerequisites
- git.
- pnpm or npm.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Grego14/taskflow
```

2. Navigate to the project directory:

```bash
cd taskflow
```

3. Install dependencies using either pnpm or npm:

```bash
pnpm install
# or
npm install
```
